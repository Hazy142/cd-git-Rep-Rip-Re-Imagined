AI GitHub Repository Analyzer

Enter a public GitHub repo URL. The AI will identify key files, provide a holistic code analysis, and optionally re-implement the project with improvements.

GitHub limits unauthenticated requests to 60/hour. Provide a PAT to increase the limit. The token is only used in your browser and is not stored.

Re-implement with Gemini

Based on the analysis, you can have Gemini re-create this project from scratch with all the suggested fixes and improvements, then download the result as a ZIP file.
