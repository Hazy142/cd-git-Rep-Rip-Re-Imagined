GitHub Repository Analyzer

Enter a GitHub repo URL. The AI will automatically identify key files, fetch them, and provide a holistic code analysis.

Unauthenticated requests are limited to 60/hour. Provide a PAT to increase the limit to 5,000/hour. The token is only used in your browser and not stored.
